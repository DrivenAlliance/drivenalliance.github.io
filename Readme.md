Driven Software Website
=======================

Website
--------

To Setup
--------
Have Node and Ruby installed.

Install Jekyll: `sudo gem install jekyll`
Install node packages: `npm install`
Generate stylesheets: `gulp style`

To Add new page
---------------

Set this tag for the layout page
<pre>
---
layout: default
---
</pre>

To Test
-------

  `jekyll serve -w`
